# EridaNorm

EridaNorm - Адаптивная нормализация с контролем дрейфа среднего для языковых моделей.

| Ситуация      | Действия                        |
| ------------- | ------------------------------- |
| mean ≈ 0      | β ≈ 0, работает как RMSNorm     |
| mean уползает | β адаптируется, стабилизирует   |
| mean огромный | сатурация √(1+β), не пережимает |


| Метод         | Проходов |  Видит mean  | fp16-safe |
| ------------- | -------- | ------------ | --------- |
| LayerNorm     | 2        | Ага          | Ага       |
| RMSNorm       | 1        | Неа          | Ага       |
| **EridaNorm** | 1        | Ага          | Ага       |


Где применять:
- Если модель < 6B параметров
- Агрессивное обучение (высокий LR, длинные контексты)
- Квантизация (меньше смещение zero-point)

| Параметр       | Дефолт | Смысл                |
| -------------- | ------ | -------------------- |
| `num_features` | —      | d\_model             |
| `eps`          | 1e-6   | от деления на ноль   |
| `beta_max`     | 0.5    | потолок β            |
| `beta_init`    | -2.2   | старт: β\_eff ≈ 0.05 |
